# PjWd-Edit-0

フロントエンドのみ（HTML/CSS/JavaScript）で動作する、プロセカ風の簡易譜面エディタです。

> このリポジトリはオリジナル実装です。既存ゲームの画像・音声・譜面データなどのアセットは含みません。

## できること

- 6レーン譜面の編集（左クリックで配置、右クリックで削除）
- ノーツの選択・ドラッグ移動
- ノーツ種別切り替え（Tap / Flick / Hold Start / Hold End）
- ホールド開始/終了ノーツのガイド線表示
- スナップ分解能（1/1〜1/16）
- BPM / 小節数 / 拍子 / ズーム編集
- Undo / Redo / Delete選択削除（ショートカット対応）
- JSON の保存・読み込み・クリップボードコピー
- ローカルストレージへの自動保存

## 起動方法

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## ショートカット

- `Ctrl/Cmd + Z`: Undo
- `Ctrl/Cmd + Y`: Redo
- `Delete`: 選択ノーツ削除
- `1`〜`4`: ノーツ種別切り替え
